Purpose

This report treats each matrix cell as a subject and analyzes three endpoints: first infection, first recovery, and death. Cells without a given event are censored at the final observed simulation step.

The event columns follow the standard R survival convention used by Surv(time, status): TRUE means the event was observed, and FALSE means the cell was censored for that endpoint.

survival_endpoints <- read_csv(file.path(derived_dir, "survival_endpoints.csv"), show_col_types = FALSE)

Cox Models

The models use row, column, distance from center, and initial infection status as cell-level covariates. Coefficients are descriptive for this single simulated run and should not be generalized beyond this baseline scenario.

fit_cox <- function(time_column, event_column) {
  event_count <- sum(survival_endpoints[[event_column]])
  if (event_count < 2) {
    return(data.frame(
      term = "not_fit",
      estimate = NA_real_,
      conf.low = NA_real_,
      conf.high = NA_real_,
      p.value = NA_real_,
      note = paste("Only", event_count, "event(s); Cox model not fit.")
    ))
  }

  formula <- as.formula(
    paste0("Surv(", time_column, ", ", event_column, ") ~ row + col + distance_from_center + initially_infected")
  )

  broom::tidy(coxph(formula, data = survival_endpoints), exponentiate = TRUE, conf.int = TRUE) %>%
    mutate(note = "Hazard ratio")
}
cox_results <- bind_rows(
  fit_cox("time_to_infection", "event_infection") %>% mutate(endpoint = "First infection"),
  fit_cox("time_to_recovery", "event_recovery") %>% mutate(endpoint = "Recovery"),
  fit_cox("time_to_death", "event_death") %>% mutate(endpoint = "Death")
) %>%
  select(endpoint, term, estimate, conf.low, conf.high, p.value, note)
